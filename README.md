🌦️ Weather App
The Weather App is a responsive web application that provides real-time weather information for any city. It fetches live data from a weather API and displays essential details such as temperature, humidity, wind speed, and current weather conditions. The project focuses on API integration, asynchronous JavaScript, and a clean user-friendly interface.
🚀 Features
Real-time weather data for any location
Displays temperature, humidity, wind speed, and weather conditions
City-based weather search
Responsive and modern UI
Error handling for invalid city names

🛠️ Technologies Used
HTML
CSS
JavaScript
Weather API
